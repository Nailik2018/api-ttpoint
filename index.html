Alle Clubs eines Verbandes

get/api/get-clubs-of-associations.php
Listet alle Clubs eines Tischtennisverbandes auf (get-all-associations.php)
Es werden alle Clubs des Verbandes aufgelistet inklusive des Schweizerischer Tischtennis Verband (STTV). Es wird der Namen des Clubs, welchem Verband dieser Club angehört, der höchste Elo Wert eines Spielers und die Anzahl der Lizenzierten Spieler Frauen und Männer zusammen als Datensatz ersichtlich sein. Es ist nur der Verband als Paramter möglich. Die Rückgabe erfolgt als JSON Array (siehe Beispiel Daten weiter unten)

Abfrage Parameter

status (required)
Query Parameter: AGTT, ANJTT, ATTT, AVVF, MTTV, NWTTV, OTTV, TTVI oder STTV

URL:

https://www.api-ttpoint.ch/api/get-clubs-of-associations.php?association=mttv

Alle Spieler eines Club

get/api/get-club.php
Listet alle Spieler einse Tischtennisclub auf (get-club.php)
Es werden alle Spieler des Clubs aufgelistet. Es werden Clubinformationen wie der Name des Clubs, zu welchem Verband der Club gehört und die Anzahl Lizenzierten Spieler angezeigt. Zudem werden die Spieler mit dem Vornamen, Nachnamen, das Gechlecht des Spielers und der Elo Wert als Datensatz ersichtlich sein. Es ist der Club als Paramter möglich. Optional kann als Paramter noch der Monat und das Jahr mitgebenen werden, Dies ermöglicht eine Zeitreise zu einem gezielten Moment des Clubs. Die Rückgabe erfolgt als JSON Array (siehe Beispiel Daten weiter unten)

Abfrage Parameter

status (required)
Query Parameter: Ostermundigen, Bern, Rio-Star Muttenz oder ZZ-Lancy

URL(s):

https://www.api-ttpoint.ch/api/get-club.php?clubname=ostermundigen
oder
https://api-ttpoint.ch/api/v1/get-club.php?clubname=ostermundigen&month=8&year=2020

Rückgabe Typ:

array[Club]

Content-Type:

  • application/json

Responses:

200

successful operation

400

Invalid status value

Status:

Abgeschlossen
