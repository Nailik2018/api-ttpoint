get/api/get-all-associations.php
Listet alle regionale Tischtennis Verbände der Schweiz auf (get-all-associations.php)
Es werden alle regionale Tischtennis Verbände der Schweiz aufgelistet inklusive des Schweizerischer Tischtennis Verband (STTV). Es sind keine Parameter per URL möglich. Die Rückgabe erfolgt als JSON Array (siehe Beispiel Daten weiter unten)
