Listet alle Schweizer Tischtennis Verbände auf (get-all-associations.php)
Dieser API Aufruf gibt alle Tischtennis Verbände der Schweiz aus. Es sind keine Parameter per URL möglich. Die Rückgabe erfolgt als JSON Array (siehe Beispiel Daten)

Rückgabe Typ:
